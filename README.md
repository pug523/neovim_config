### Installation

Clone this repository into your Neovim configuration directory:  
**If you have an existing Neovim configuration, back it up first.**  

```bash
git clone https://github.com/pug523/neovim_config.git ~/.config/nvim
```

### Requirements

- nvim v0.12+ 

### Directory structure

- `init.lua`: entry point
- `lua/options.lua`: editor options
- `lua/mappings.lua`: keymaps
- `lua/diagnostic.lua`: diagnostic handling
- `lua/style.lua`: color schemes and appearance
- `lua/lsp/`: language servers configuration
- `lua/plugins/`: plugin management and configuration
